11.10. 小结

openanything.py 及其函数现在可以完美地工作了。

每个客户都应该支持的 5 个 HTTP web 服务重要特性:

  • 通过设置适当的 User-Agent 识别你的应用。
  • 处理 适当的永久重定向。
  • 支持 Last-Modified 日期检查 从而避免在数据未改变的情况下重新下载数据。
  • 支持 ETag hash 从而避免在数据未改变的情况下重新下载数据。
  • 支持 gzip 压缩 从而在数据 已经 改变的情况下尽可能地减少传输带宽。
